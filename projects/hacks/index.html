Gmail and libgmail

I was recently faced with a problem of downloading my Google mail using Web interface only (POP3/IMAP and forwarding were disabled).

I found a project (a library) called libgmail.py that does just what is needed. Unfortunately, Google has changed Web interface and the tool does not work anymore. The project at this point is abandoned as far as I can tell.

I wrote a Python module (Gmail.py) that deals with Google's Single Sign On solution and retrieves messages using Gmail Web interface. The second module (GmailFetchmail.py) retrieves messages using Gmail.py and feeds them to the local sendmail MTA to be received using normal UNIX mail tools. The source code can be found here.

Before using Gmail.py you almost certainly would need to modify the source to account for your particular Single Sign On form details (the Gmail.login() method). You also need to switch Gmail into Basic HTML mode.
